## 群内指令 
> 请在群内直接发送指令消息以使用  
  示例:
![示例](7th/images/assets/command-in-qqgroup-list.png)

**绑定Xbox ID(获取白名单)**  
`/bind <你的id>`  
*不要加尖括号*

**查询在线玩家列表**  
`/list`

**查询服务器状态**  
`/stat`  
注:如果返回"查询失败 请检查服务器是否开启"则表明服务器此时已关闭或无法连接，请反馈给腐竹.
