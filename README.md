# xyzmc-doc
Docs for A Minecraft Server.

# 登录插件相关
**注册**  
`/register <密码> <重复密码>`  
或 `/reg <密码> <重复密码>`

**登录**  
`/login <密码>`  
或 `/l <密码>`

**修改密码**  
`/changepassword <旧密码> <新密码> <重复新密码>`  
或 `/changepw <旧密码> <新密码> <重复新密码>`

**绑定邮箱**   
`/bindemail set <邮箱>`  
或`/bdmail set <邮箱>`
